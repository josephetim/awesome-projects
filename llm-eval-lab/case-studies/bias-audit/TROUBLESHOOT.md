# TROUBLESHOOT

## High variance between runs

- Keep temperature low for reproducible comparisons.
- Run multiple trials and average disparity scores.

## API key/provider errors

- Ensure `.env` contains the correct key for selected provider.
- Verify `PROVIDER` and selected dashboard provider are aligned.

## Interpreting disparity safely

- A high score indicates potential inconsistency, not definitive bias proof.
- Perform manual review before making policy conclusions.
