# EXTEND Guide

## Extension Ideas

1. Add larger demographic and occupational prompt libraries.
2. Add classifier-based toxicity and sentiment checks.
3. Add confidence intervals via repeated runs per prompt pair.
4. Add human review labels for flagged high-disparity cases.

## Practical Advice

- Automated disparity heuristics are directional signals, not legal conclusions.
- Combine model-based audits with domain and policy review.
