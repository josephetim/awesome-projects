# CTF Writeups Submission Guide

This directory accepts safe, legal writeups only.

## Allowed Writeups

- CTF challenges from authorized platforms
- Local lab challenge walkthroughs
- Defensive lessons learned from challenge solutions

## Required Format

1. Challenge name and platform
2. Legal/authorization context
3. Problem statement
4. Step-by-step solution
5. Defensive takeaway
6. References (if used)

## Not Allowed

- Real target identifiers
- Unauthorized exploit instructions against live systems
- Sensitive leaked data
