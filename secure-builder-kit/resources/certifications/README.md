# Security Certification Roadmaps

## Security+

### Focus
- Core security concepts
- Network and system security fundamentals
- Risk management and incident response basics

### Roadmap
1. Learn networking and system basics.
2. Study threats, attacks, and vulnerabilities.
3. Practice log analysis and security operations labs.
4. Run timed practice exams weekly.

## CEH

### Focus
- Ethical hacking methodology
- Recon, scanning, exploitation concepts
- Defensive countermeasures

### Roadmap
1. Build legal lab setup and document scope boundaries.
2. Practice each kill-chain stage in isolated labs.
3. Study common tools and defensive mitigations.
4. Focus on exam-style scenario interpretation.

## eJPT

### Focus
- Practical penetration testing in controlled environments
- Enumeration, exploitation, post-exploitation basics
- Report writing

### Roadmap
1. Master Linux, networking, and scripting basics.
2. Practice network and web enumeration labs daily.
3. Perform full attack-path labs and write reports.
4. Review privilege escalation and pivoting essentials.

## OSCP

### Focus
- Advanced practical penetration testing
- Realistic chained exploitation
- Strong documentation under time pressure

### Roadmap
1. Strengthen scripting (Python/Bash/PowerShell).
2. Build deep AD, web, and Linux exploitation workflows.
3. Practice long-form labs with strict time boxing.
4. Improve report quality and reproducibility evidence.
