# ETHICS AND LEGAL BOUNDARIES

This repository is for educational and defensive security training only.

## Allowed Targets

- Your own local lab systems
- DVWA instances you control
- Intentionally vulnerable demo apps included in this repository
- Systems where you have explicit written permission to test

## Prohibited Use

- Unauthorized scanning, exploitation, or service disruption
- Testing production systems without documented approval
- Any activity that violates laws, contracts, or acceptable-use policies

## Safe Practice Principles

1. Minimize harm: test in isolated lab environments.
2. Document intent: keep notes on scope and authorization.
3. Prefer defense: always pair attack demonstrations with mitigation steps.
4. Report responsibly: disclose findings through approved channels.

## Operator Responsibility

You are solely responsible for legal compliance and ethical conduct.
